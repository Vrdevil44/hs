## üåê How to Deploy to GitHub Pages

1.  **Push to GitHub**:
    - Initialize a git repo (if not done): `git init`
    - Add files: `git add .`
    - Commit: `git commit -m "Initial commit"`
    - Create a repo on GitHub and add remote: `git remote add origin <your-repo-url>`
    - Push: `git push -u origin main`

2.  **Enable GitHub Pages**:
    - Go to your repository **Settings**.
    - Click on **Pages** in the left sidebar.
    - Under **Build and deployment**, select **Source** as `Deploy from a branch`.
    - Select `main` branch and `/ (root)` folder.
    - Click **Save**.

3.  **View Your Site**:
    - After a few minutes, GitHub will provide a URL (e.g., `https://yourusername.github.io/my-portfolio/`).
